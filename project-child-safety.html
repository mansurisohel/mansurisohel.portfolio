Smart Child Safety System

An IoT-based safety monitoring system with SOS alerts, GPS tracking, and real-time notifications for enhanced child security.

Technologies Used

IoT Sensors GPS Module wokwi simulation

Key Features

  • Real-time GPS location tracking
  • SOS emergency button
  • Instant parent notifications
  • Safe zone alerts
  • Battery status monitoring

Project Purpose

Designed to provide parents peace of mind by enabling real-time monitoring of their child's location and safety. The system sends instant alerts in emergency situations and allows parents to define safe zones, receiving notifications when boundaries are crossed.
